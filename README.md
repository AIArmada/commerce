**AIArmada Commerce is a modular e-commerce stack for Laravel 13 — carts, payments, vouchers, shipping, inventory, and Filament admin panels in one cohesive toolkit.**

Stay focused on product logic while Commerce handles pricing rules, concurrency, webhooks, and polished back-office tooling.

[Read the documentation](docs/index.md) • [Installation guide](INSTALLATION.md) • [Join the discussions](https://github.com/aiarmada/commerce/discussions)

```bash
composer require aiarmada/commerce
php artisan commerce:setup
php artisan migrate
```

Need a package-by-package install path instead of the suite? Start with [INSTALLATION.md](INSTALLATION.md) and the canonical package docs under `packages/*/docs/`.
