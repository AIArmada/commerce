# Repo Memories

`memories/` is assistant working memory for this repository.

## Important boundaries

- Memories are **not** canonical product or package documentation.
- Durable behavior should live in `packages/*/docs` or active root docs under `docs/`.
- Time-boxed audits, exploratory notes, and superseded findings should move to `memories/archive/` or be deleted once captured elsewhere.

If an AI assistant sees both docs and memories, it should prefer the docs unless the task is explicitly about previous assistant findings.
