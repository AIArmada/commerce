# Filament CHIP

Filament admin plugin for exploring CHIP payment data.

## Quick Start

```bash
composer require aiarmada/filament-chip
```

```php
use AIArmada\FilamentChip\FilamentChipPlugin;

$panel->plugin(FilamentChipPlugin::make());
```
