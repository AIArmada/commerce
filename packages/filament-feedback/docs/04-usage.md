# Usage

## Manage forms

Navigate to the Feedback Forms resource. Create a new form with name, purpose, visibility, and scheduling. Add sections and questions using the relation managers.

## Use the builder

The form builder page allows adding, editing, reordering, and deleting sections, questions, and options. Validation rules and branching visibility can be configured per question.

## Send invitations

From the invitations relation manager on a form, create and send invitation links. Track sent, opened, started, and submitted states.

## Review responses

View submitted responses with status badges. Review, reject, or mark responses as spam. Export responses to CSV.

## Moderate testimonials

Navigate to the Testimonials resource. Approve, reject, publish, or hide extracted testimonials. Published testimonials require explicit permission.

## Dashboard analytics

The feedback dashboard provides widgets for:
- Overview stats (total forms, responses)
- Response trend chart
- Average rating
- NPS score
- CSAT score
- Rating distribution
- Latest comments
- Completion rate
- Pending testimonials

## Export responses

Use the built-in Filament export actions to export responses, answers, and testimonials. All exports are owner-scoped.
